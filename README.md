# Java Backend Fundamentals

This repository documents my process of refreshing, strengthening, and expanding my Java fundamentals as preparation for backend development.

I have previously learned the basics of Java, and I am now revisiting the core concepts in a more structured and practical way before moving deeper into Spring Boot, SQL, REST APIs, testing, Docker, and backend architecture.

## Topics Covered

* Object-Oriented Programming
* Encapsulation
* Inheritance
* Polymorphism
* Abstraction
* Interfaces and abstract classes
* Casting
* `equals()` and `hashCode()`
* Primitive types and wrapper classes
* Strings
* Collections
* Exceptions
* Generics
* Lambdas and Streams
* Mini Java projects

## Goal

The goal of this repository is to build a strong and reliable Java foundation for backend development.

Each topic contains small practical examples, notes, and exercises that help reinforce important Java concepts used in real-world backend applications.

## Next Steps

After strengthening Java fundamentals, the next focus will be:

* Spring Boot
* PostgreSQL
* REST API development
* Testing with JUnit and Mockito
* Docker
* GitHub Actions
* Backend project architecture
